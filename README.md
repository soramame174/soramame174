- 音楽クリエイター同士がフリーの楽曲を登録・検索・共有できるプラットフォームです。
- 詳細は上記リポジトリのREADMEをご覧ください。

私が開発した従業員勤怠管理・生産性向上WEBアプリです。
### [勤怠管理・業務効率化システム](https://github.com/soramame174/attendance_system)
- 複数のITベンチャー企業での利用を想定した、従業員の勤怠管理業務の効率化と生産性向上を支援する統合管理プラットフォームです。
**管理者向け機能:**
- 勤怠状況のリアルタイム監視と詳細グラフ生成（月別労働時間・出勤日数）。
- 社員情報の管理（登録・編集・削除）およびセキュリティを考慮した専用コード発行機能。
**従業員向け機能:**
- ToDoリストによるタスク管理と重要度設定。
- チーム連携を強化する共有カレンダー機能（日数を跨いだ予定登録に対応）。
- 特徴: 既存の管理ツールにおける「操作の複雑さ」や「フォロー不足」という課題を解決し、直感的なUIで管理者の負担を大幅に削減します。
